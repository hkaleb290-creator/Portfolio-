# Portfolio Starter Template

This folder is a reusable version of your portfolio that other people can copy and personalize.

## Quick start

1. Copy this `portfolio-starter-template` folder.
2. Open `index.html`.
3. Replace all placeholders in this format: `{{LIKE_THIS}}`.
4. Add your own resume file and update links.
5. Deploy on GitHub Pages, Netlify, or Vercel.

## Placeholder checklist

- `{{YOUR_NAME}}`
- `{{YOUR_FIELD_OR_TITLE}}`
- `{{ONE_LINE_DESCRIPTION}}`
- `{{YOUR_EMAIL}}`
- `{{RESUME_FILE_OR_LINK}}`
- `{{GITHUB_OR_LINKEDIN_LINK}}`
- `{{PROJECT_COUNT}}`
- `{{FOCUS_AREA}}`
- `{{CITY_OR_REGION}}`
- Project cards (`{{PROJECT_1_*}}`, `{{PROJECT_2_*}}`, `{{PROJECT_3_*}}`)
- Timeline (`{{DATE_RANGE_*}}`, `{{ROLE_*}}`, `{{ORG_*}}`, `{{ROLE_*_SUMMARY}}`)

## Optional improvements

- Add a profile image in the hero section.
- Add a dedicated projects page for long case studies.
- Add analytics (Plausible, Google Analytics, etc.).
- Add a contact form (Formspree, Basin, etc.).

## Local preview

From the main workspace root, run:

```bash
node server.js
```

Then open: `http://localhost:8000/portfolio-starter-template/`

## Publish to Jorge GitHub repo

From the workspace root, run:

```bash
publish-jorge.cmd
```

Optional custom commit message:

```bash
publish-jorge.cmd Update project cards and resume link
```
